Estimote Unity Indoor Location

Indoor Location positioning system for Unity

What is Estimote Unity Indoor Location?

The Estimote Unity Indoor Location plugin allows you to use the Indoor Location SDK within Unity and deploy to iOS or Android. It gives you positional information within a location that has been setup using the Indoor Location SDK.
