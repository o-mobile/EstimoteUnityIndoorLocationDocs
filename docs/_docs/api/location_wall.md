This class represents a LocationWall from the Estimote Indoor Location SDK.

### Variables
* X1 (double)
  * The walls X1 value.
* Y1 (double)
  * The walls Y1 value.
* X2 (double)
  * The walls X2 value.
* Y2 (double)
  * The walls Y2 value.
* Orientation (double)
  * The walls orientation value.
